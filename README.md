# Dynamic Tree Builder
A lightweight, no-backend web application designed to help you organize ideas, create mind maps, or manage your personal knowledge base in a clean, hierarchical tree structure. 

Inspired by the clean aesthetics of Google's Material Design 3 (NotebookLM style), this tool works entirely in your browser.

## ✨ Key Features
* **Infinite Hierarchy:** Create branches and sub-branches with no depth limits.
* **Material Design UI:** Clean, modern interface with smooth animations, rounded connecting lines, and distraction-free viewing.
* **Rich Node Customization:** Add descriptions to branches (hidden behind a sleek toggle) and customize branch tag colors.
* **Lightning-Fast Search:** Instantly filter your tree. The search engine checks both titles and descriptions, automatically expanding relevant branches.
* **Dark / Light Mode:** Fully supported dark theme that remembers your preference.
* **100% Private (No Database):** All your data belongs to you. Export your entire tree as a local `.json` file and load it back whenever you need it.

## 🚀 How to Use
Since this application runs entirely on the client side (perfect for GitHub Pages), it doesn't use a cloud database. 
1. **Build:** Start adding branches, naming them, and picking colors.
2. **Save:** Click **"💾 Save JSON"** before closing the tab to download your tree as a `.json` file.
3. **Load:** Next time you open the app, simply click **"📂 Load JSON"**, select your file, and continue exactly where you left off.
